## 🌟 About This Repo

<img align="right" width="300" src="https://media.giphy.com/media/qgQUggAC3Pfv687qPC/giphy.gif"/>

> 🎯 A **beginner-friendly** collection of Jupyter notebooks where I explored Python's most powerful data libraries — one concept at a time.

- 📌 **No fluff** — just clean, focused notebooks
- 📂 **Real datasets** — CSV, JSON, Kaggle data
- 🔰 **Basics only** — perfect starting point
- 💡 **4 Libraries** — NumPy · Pandas · Matplotlib · Seaborn
